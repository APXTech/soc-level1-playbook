A SOC Level 1 analyst is responsible for continuous monitoring, initial alert triage, log analysis, and incident reporting.

Key responsibilities include:
- Monitoring security alerts from SIEM tools
- Performing initial analysis and validation
- Classifying incidents based on severity
- Escalating incidents to higher levels
- Preparing incident reports and documentation

---

## üîç Log Monitoring & Analysis

Common log sources:
- Firewall logs
- Windows Event Logs
- Linux system logs
- Authentication logs
- Application logs

Basic analysis steps:
1. Identify the alert source
2. Review timestamp, source IP, destination IP
3. Check event type and severity
4. Correlate with past events
5. Determine false positive vs real incident

A standard incident report includes:

- Affected systems
- Indicators of compromise (IOC)
- Initial analysis
- Action taken
- Escalation status
